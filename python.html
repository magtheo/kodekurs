Velkommen til Python-verdenen!

Python er et utrolig populært og nybegynnervennlig programmeringsspråk som brukes til alt fra nettsider og apper til spill og kunstig intelligens.

I dette kurset vil du lære det grunnleggende i Python, slik at du kan skrive dine egne små programmer og se koden din komme til live!

Vi starter med å forstå hva programmering er, og bygger deretter steg for steg med variabler, løkker, betingelser og funksjoner.

Hva lærer du?

  • Hva Python er og hvorfor det er så populært
  • Slik skriver du dine første linjer med kode
  • Variabler for å lagre informasjon
  • Bruk av input og output
  • If-setninger for å ta beslutninger
  • For- og while-løkker for å gjenta ting

For å komme i gang trenger du en Python-tolk og en enkel tekstbehandler. Vi anbefaler å bruke en online editor som Replit, eller å installere Visual Studio Code med Python-utvidelsen på din egen datamaskin.
